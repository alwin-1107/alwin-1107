## 👋 About Me

I'm an **Information Technology** undergrad at FCRIT, Navi Mumbai (class of 2026), building systems that sit at the crossroads of **machine learning**, **computer vision**, and **real-time web engineering**. My Final Year Project is a mobile-first AR measurement app that runs entirely in the browser — no ARKit, no ARCore.

- 🔭 Currently building **RoomScan** — 3D object measurement + WebAR visualization on Android
- 🧠 Interested in CV, NLP, distributed pipelines, and browser-native ML
- 🌐 Everything I ship runs on open-source, no proprietary SDKs required
- 📍 Mumbai, India
